JS Degisken Tipleri


JS'de saklanabilen 5 adet veri tipi vardir:
  • string
  • number
  • boolean
  • object
  • function


    • JS'de 3 farkli nesne turu vardir:
  • object
  • date
  • array


    • JS'de deger icermeyen 2 degisken turu vardir:
  • null
  • undefined


    • JS'de Tip Donusumleri:
  • sayi -> metin : String(number)
  • sayi -> metin : number.toString()
  • boolean -> metin : String(bool)
  • boolean -> metin : bool.toString()
  • date -> metin : String(date)
  • date -> metin : date.toString()
  • metin -> sayi : Number(metin)
